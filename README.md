Smart Voting System

The Smart Voting System is a secure and user-friendly web-based application designed to simplify and digitize the voting process. It ensures transparency, prevents duplicate voting, and provides real-time vote tracking.

 Features
User Authentication:
Secure login system for voters using username and password.
Candidate Selection:
Displays a list of candidates for users to choose from.
One Person, One Vote:
Prevents multiple voting by the same user.
Real-Time Vote Counting:
Votes are stored in a database and updated instantly.
Interactive UI:
Built using an intuitive interface (e.g., Streamlit) for a smooth user experience.
